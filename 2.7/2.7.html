2.7 Sidlayout

Man kan använda sig av olika sidlayouts som kan få sidan att bete sig på ett annorlunda sätt när man förstorar/förminskar fönstret. Sidlayouts är olika vilket man kan anpassa basserad på hur fönstret på webbläsarens storlek. Det finns tre olika

Fixerad layout detta innebär att sidan inte kommer att anpassa sig efter storleken på webbläsarfönstret eller storleken på de interna elementen. layouten på sidan blir då fast, vilket innebär att man inte ens kan zooma in etc.

Flytande layout syftar på att sidan anpassar sig med hjälp av enheten: "%".

Elastisk layout Vilket innebär att layouten anppasar sig till övriga element, vilket då sker med enheten: "em".

Varför man ska använda CSS och inte tabeller för detta?

Sidan laddar snabbt och svarar snabbare och bättre om man använder sig utav CSS istället för tabeller.
Om man använder tabeller så har man sin "stil" kod blandat med innehållet vilket gör att koden blir längre och klumpigare.
Man kan även kontrollera utseendet i dokumentets layout mer med hjälp utav CSS. Om man använder sig av tabeller
så måste man ändra layouten i varje tabell i varje sida.
