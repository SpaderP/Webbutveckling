Jag använder här float samt clear. Float: för att bestämma hur helementet ska placera sig, man kan även använda float för att få element att placera sig bredvid varandra. Clear: med clear gör jag så att det elementet som jag använder clear på placeras efter float elementet.

Jag använder även Height och Width för att bestämma höjd och bredd på ett objekt.Max-height, Min-height, Max-width samt Min-width används för att sätta den minsta/största bredden samt höjden på elementet.


Position anger hur ett element/objekt ska positioneras. Detta kan vara statisk som är det normala. När ett element har relativ positionering så är dens position relativ till den standardpositionen. Absolut innebär att dens position är relativ till dens närmaste-positionerade släkting. Fixerad är den då dens positon är relativ till fönstret och är alltid på samma plats på fönstret va du än gör som användare. Sticky är relativt till användarens scroll-position.


Denna div har position: fixed.


Z-index

Ju högre z-index värde ett element har ju högre lager ligger elementet på. Det elementet som ligger på högre lager kommer att täcka elementen på lägre lager.


Z-index på bilden är -1.
